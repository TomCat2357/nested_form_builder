## 追加撮影が必要な画面

### 優先度：高
1. **設定画面**
   - アプリケーション設定
   - デフォルト値の設定

2. **質問追加・編集画面**
   - 質問タイプの選択
   - 質問の詳細設定
   - ネスト（階層化）の操作
   - 質問の並び替え

3. **プレビュー画面**
   - 作成したフォームのプレビュー表示

### 優先度：中
4. **検索画面**
   - 検索クエリ入力
   - 検索結果の表示
   - 高度な検索機能（AND/OR、比較演算子など）

5. **フォーム編集画面（既存フォーム）**
   - 保存済みフォームの編集状態

6. **生成されたHTMLフォーム**
   - スタンドアロンで動作するフォームの表示例
   - 実際の入力例

### 優先度：低
7. **データ管理画面**
   - 回答データの一覧表示
   - データの編集・削除

8. **エクスポート/インポート画面**
   - フォーム定義のエクスポート
   - フォーム定義のインポート

9. **エラー表示例**
   - バリデーションエラー
   - 保存エラー

## 画像挿入の手順

### Phase 1: スクリーンショット撮影
1. Webアプリに戻る
2. 各機能の画面を順次表示
3. native-browser-controlのscreenshotメソッドで撮影
4. ファイル名を分かりやすく命名（例: `screenshot_05_settings.png`）

### Phase 2: マニュアルへの挿入
1. Googleドキュメントのマニュアルを開く
2. 各セクションに対応する画像を挿入
   - 方法1: クリップボード経由で貼り付け
   - 方法2: Googleドキュメントの「挿入」メニューから画像をアップロード
3. 各画像にキャプション（説明文）を追加

### Phase 3: 説明文の追加
1. 画像の前後に操作手順を記述
2. 重要なボタンや機能を強調
3. 注意事項や Tips を追加
