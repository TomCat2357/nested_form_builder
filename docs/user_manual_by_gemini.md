# 🔰 自分専用フォーム作成ツール かんたん操作マニュアル

このツールは、配られたURLにアクセスするだけで、**あなたのGoogleアカウントに紐付いた「あなた専用のフォーム作成・管理画面」**が開くシステムです。

- あなたが作ったフォームは他の人からは見えません（他の人が作ったフォームもあなたからは見えません）。
- データはすべて **あなた自身のGoogleドライブ** に保存されます。

安心して、自分だけのアンケート、申請フォーム、出欠確認ツールなどを作成してください。

---

## 📌 ステップ1：新しいフォームを作ろう

### 1. フォームの作成画面を開く
1. ツールにアクセスして最初の画面（メイン画面）が開いたら、左側メニューから **[フォーム管理]** をクリックします。
2. 左メニューの **[新規作成]** ボタンをクリックします。

### 2. 基本情報を設定する
- **フォーム名・説明**  
  「〇〇部 懇親会 出欠確認」など、分かりやすいタイトルと説明を入力します。
- **フォーム項目データのGoogle Drive保存先URL**  
  ここは **空欄のままで大丈夫** です。空欄にしておくと、回答が自動的にまとまる表（Googleスプレッドシート）が、あなたのGoogleドライブのルート（一番上の階層）に自動で作成されます。
- **（オプション）アクセス制御**  
  「No.を表示する」「自分の回答のみ表示」などの設定も可能です。最初は初期設定のままで問題ありません。

### 3. 質問を作る
画面下部の **[質問を追加]** ボタンを押して、質問を作ります。

- **項目名を入力**：質問のタイトルを入力します。
- **種類を選ぶ**：右側のプルダウンから質問の種類を選びます。  
  - `テキスト` / `テキスト（複数行）`：自由記述  
  - `ラジオ` / `ドロップダウン`：1つだけ選ぶ  
  - `チェックボックス`：複数選べる  
  - `日付` / `時間`：カレンダーや時計から入力  
  - `入力ユーザー名`：回答者のGoogleアカウントのメールアドレスを自動取得  
  - `メッセージ`：回答欄はなく、案内文だけを表示する
- **「必須」チェック**：必ず答えてほしい質問にはチェックを入れます。
- ⚠️ **「表示」チェック（重要）**  
  チェックを入れると、後で回答を一覧表示する画面で、この質問の答えが **「列」** として表示されます。  
  パッと見て確認したい重要な質問（氏名など）は、必ずチェックを入れてください。

### 4. 条件つき質問（枝分かれ）を作る
「『はい』と答えた人にだけ次の質問を表示する」などの設定も簡単です。

1. 質問の種類を `ラジオ` / `ドロップダウン` / `チェックボックス` のいずれかにします。
2. 選択肢（例：「はい」「いいえ」）を作ります。
3. 選択肢の右側にある **[子質問追加]** を押します。
4. 選択肢の下に、少しズレた形で新しい質問枠が追加されます。  
   これが「その選択肢を選んだ人にだけ表示される質問」です。

### 5. プレビューで確認して保存する
- 画面上部の **[プレビュー]** タブで、回答者の見え方を確認できます。確認後は **[編集]** タブに戻れます。
- 作り終えたら、左側メニューの **[保存]** をクリックして完成です。

## 📌 ステップ2：回答用URLを配って、回答を入れてもらおう

### 1. URLをコピーする
1. 左側メニューの **[フォーム管理]** をクリックし、フォーム一覧に戻ります。
2. 目的のフォームの右側（フォームIDの列）にある **[📋（コピーマーク）]** をクリックします。  
   チェックマーク「✓」に変わればコピー完了です。

### 2. URLを配る
- コピーしたURLを、メールやチャットで回答者へ送信します。
- このURLを開いた人には、管理画面や他の人のデータは見えず、**「回答する画面だけ」**が表示されます。

### 3. 自分でテスト入力してみる
1. メイン画面（フォーム一覧）で、作ったフォームの四角いパネルをクリックします。
2. 左メニューの **[新規入力]** を押して回答画面を開きます。
3. 実際に入力し、画面下部の **[保存]**（または設定したボタン名）を押してみてください。

## 📌 ステップ3：集まった回答を確認・修正・検索しよう

### 1. 回答一覧（検索画面）を開く
1. 画面左上のタイトル（または「戻る」ボタン）でメイン画面に戻ります。
2. 確認したいフォームの四角いパネルをクリックします。
3. **検索画面**が開き、回答が表（テーブル）で表示されます。  
   - 表の項目（列見出し）をクリックすると並び替え（昇順・降順）ができます。

### 2. キーワードで検索・絞り込み
- 画面上部の **「キーワード検索」** 欄に、探したい言葉（氏名など）を入力します。
- 入力するだけで、該当する回答だけに絞り込まれます。
- *一覧に列が出ない場合*：フォーム編集で、該当質問の **「表示」** がオフになっている可能性があります。チェックを入れて保存し直してください。

### 3. 回答の確認・修正・削除
- **詳細を見る／修正する**：表の行（回答）をクリックすると詳細画面が開きます。左メニューの **[編集]** から修正して保存できます。
- **削除する**：一覧表の左端チェックボックス（☑️）で対象行を選び、左メニューの赤文字 **[削除]** を押します。

### 4. スプレッドシートで全データを見る
- 回答は Googleスプレッドシート（Google版のExcel）にも自動で記録されています。
- 左メニュー下部の **[📊 スプレッドシートを開く]** をクリックすると、保存先の表が別タブで開きます。  
  集計・グラフ作成・印刷なども可能です。

## 📌 ステップ4：不要になったフォームを整理しよう

### アーカイブ（隠す）
- 終わったイベントのフォームなどは、**[フォーム管理]** で対象フォームにチェックを入れて **[アーカイブ]** を押します。  
  データは消えずに、メイン画面のリストから非表示にできます。

### 削除（ゴミ箱）
- 完全に不要になったフォームは、同様にチェックを入れて **[削除]** を押します。  
- ※システム上からは見えなくなりますが、**Googleドライブ上のスプレッドシート本体は自動削除されません**。不要な場合はドライブから直接削除してください。

## 💡 ちょっと便利な応用機能

### 画面の色（テーマ）を変える
- 左メニューの **[設定]** からテーマ（色合い）を変更できます。  
  例：Sakura（ピンク系）／Ocean（青系）／Dark（黒背景） など

### 文字の大きさや色を変える（スタイル設定）
- 質問作成画面で「スタイル設定」にチェックを入れると、文字を大きくしたり色を変えたりできます。  
  重要な注意事項を目立たせたいときに便利です。

### フォームをコピーして再利用する（エクスポート／インポート）
- **[フォーム管理]** でフォームにチェックを入れ **[エクスポート]** を押すと、フォーム構造をファイルとしてダウンロードできます。
- **[インポート]** を使えば、過去に作ったフォームを取り込み、再利用できます。
